📋 项目概览

项目背景

本项目基于 OPS 3.5 (Open Preprint Systems) 搭建学术预印本网站平台，并集成 Artalk 评论系统实现文章评论功能。

核心功能

  • ✅ OPS 3.5 预印本系统部署
  • ✅ Artalk 评论系统集成
  • ✅ 评论插件开发（读取OPS用户状态）
  • 🔄 中文本地化完善
  • 🔄 系统配置优化
  • 🔄 用户协议和版权声明

技术栈

  • 主系统: OPS 3.5 (PHP-based)
  • 评论系统: Artalk (self-hosted)
  • Web服务器: Apache2
  • 部署环境: Linux VPS + Docker
