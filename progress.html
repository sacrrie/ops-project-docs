📝 如何更新进度

更新方法（推荐）：

  1. 在本地编辑 progress.html 文件
  2. 修改对应任务的状态、复选框等
  3. 使用Git提交并推送： 
    git add progress.html
git commit -m "更新任务进度"
git push
  4. 等待1-2分钟，GitHub Pages自动更新

备选方法：

直接在GitHub网页界面编辑 progress.html 文件，点击Commit changes即可。
